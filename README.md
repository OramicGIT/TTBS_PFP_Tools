# TTBS .PFP Tools
Tools to pack/unpack PFP files in "The Tiny Bang Story"

# How to build
Download and install .NET 8, and download this repo.

1. cd /path/to/downloaded/repo (Enter the path where you downloaded the project)
2. dotnet build (in the folder you just entered)

Go to /path/to/downloaded/repo/Bin/Debug/ and your builded code should be here.
